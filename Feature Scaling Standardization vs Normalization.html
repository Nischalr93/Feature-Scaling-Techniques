Feature Scaling Standardization vs Normalization¶

Feature scaling is an integral part of feature engeenering.

It is not neessary that we use feature scaling in each and every algorithm.

The two scales that we use is:¶

1) Normalization¶

2) Standardization¶

Normalization :¶

normalization helps you to scale down your feature from 0 to 1.

Standarization :¶

standardization helps you to scale down your feature based on standard normal distribution. Over here the mean is usually 0 and standardeviation is usually 1.

When to use Standardization and Normalization?¶

Whenevr you are using some machine learning algorithms which involves "Euclidean distance" and suppose in some of the deep learning techniques where "Gradient Distance" is basicaly involved. "Gradient Distance means where the parabola curve where you need to find the best/global minimal point OR in order to retrieve that minimal point we have to scale down that particular values."
